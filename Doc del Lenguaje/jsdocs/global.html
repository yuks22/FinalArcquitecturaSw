connect()

Función que se encarga de abrir una conexión a "/gs-guide-websocket". Si la conexión es exitosa, el cliente se subscribe a la destino "/topic/greetings", en donde el server estará publicando los mensajes. Cuando se reciba el mensaje, se agregará un elemento de párrafo al DOM para mostrar el mensaje.
